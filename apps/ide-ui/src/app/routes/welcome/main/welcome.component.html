Welcome to DontCode !

How to play with it ?

You can check you are connected to the online service by looking at the icon in the header panel.

means you are ok, you have connection issue to the online service. Please check the browser log in this case

If , then open the preview by clicking on .It should open a new tab with the Preview engine. As well, the preview engine connects to online services

Then select the left menu "Editor" and start filling values and see the Preview application reacting to it in realtime !

Can I see something ?

Of course ! This demo version is still very limited but it can show you the principle.Over the time, the functionality of DontCode will increase as Plugins are added

How does it work ?

You are running the Application Editor part, it connects to an online servicet hat receives updates as you make them and dispatch the updates to the Preview Engine

Dontcode will be a complete application builder

Showing you in realtime a preview of your application as you build it. You build it by answering questions about what you want the application to do
